Implementation of the kNN algorithm, with the data being read from .csv files and being output into a new prediction .csv file. A confusion matrix is produced alongside various details like accuracy, sensitivity, precision and specificity.

To run:
Compile row.java and KNearest_neighbour.java files and run KNearest_neighbour.

The input files are breast_cancer_test and breast_cancer_train.

There are also 5 .csv files available to see the results of different kNN values ranging from 1 to 9 and also pictures in the pictures folder to show the output.
